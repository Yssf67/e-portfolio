Project 01 · SAÉ2.01

Building a network for a small organisation

AdministerAC11.03AC11.04AC11.05

Project context

This project involved designing and configuring a small organisation's network in GNS3. The topology contains several VLANs, Cisco routers and switches, client workstations and Debian servers for development, production, DHCP and backup.

Work shown by the files

The project files include the subnetting plan, VLANs 100, 200, 300 and 999, trunk links, spanning tree, RIP routing and gateways. Debian servers use static addressing and services matching their assigned roles.

Skills acquired

AC11.03 - Configure basic local network functions. Creating VLANs, access ports, trunks and routing taught me how to segment and interconnect a network.

AC11.04 - Understand operating systems. Debian network configuration and DHCP, Apache, SSH and rsync services connect system administration to the overall network.

AC11.05 - Identify network failures. IOS verification commands and connectivity tests help locate addressing, interface and routing errors.

Difficulty and progress

The main difficulty was maintaining a consistent addressing plan across VLANs, point-to-point links and servers. A reliable method is to centralise the addresses, configure one link at a time and validate each stage before adding routing and services.

Project 04 · Practical work

Building an RJ45 network cable

ConnectAC12.03CE2.02CE2.03

Project context

The task was to build a Category 6 Ethernet cable and female RJ45 keystone jacks, then document the procedure. The TIA-568B standard defines the exact order of the eight conductors.

Work completed

I prepared the twisted-pair cable, stripped the outer jacket, arranged the conductors according to the colour code, crimped the connectors and wired the keystone jacks. The report records the standard, diagrams and components.

Skills acquired

AC12.03 - Deploy transmission media. The practical work helped me understand copper cable structure and the importance of preserving pairs.

CE2.02 - Apply a scientific approach. Cabling must be reproducible and checked against a standard.

CE2.03 - Select appropriate technology. Cable and connector categories depend on the expected speed and installation.

Difficulty and progress

Positioning the small conductors without untwisting the pairs requires accuracy. Checking the colour order twice before crimping prevents rework and improves physical link quality.

Project 06 · WebRadio

Controlling a connected radio with an ESP32

DevelopConnectAC13.02AC13.03CE2.03

Project context

The WebRadio project connects an ESP32 to Wi-Fi, a VS1053 audio decoder and an MQTT broker. Users can change stations, adjust sound settings and monitor the system state.

Work shown by the sources

The program configures MQTT topics, reconnects when needed, initialises Wi-Fi and the decoder, processes remote commands and maintains audio streaming in the main loop.

Skills acquired

AC13.03 - Translate an algorithm for a specific environment. Functions control hardware components and a network protocol on a microcontroller.

AC13.02 - Read and correct a program. Serial diagnostics distinguish Wi-Fi, MQTT, SPI and decoder failures.

CE2.03 - Select suitable technologies. MQTT is appropriate for lightweight commands and state messages between connected devices.

Difficulty and progress

This system combines several possible sources of failure. The effective method is to validate Wi-Fi, MQTT, SPI and the decoder independently before integrating audio streaming. Serial messages then become diagnostic evidence.
